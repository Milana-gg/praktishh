# Модель сегрегации Шеллинга

## Описание
Программа реализует модель сегрегации Шеллинга с визуализацией.

## Используемые технологии

- Python 3.12.6
- NumPy — для работы с массивами и сеткой
- Matplotlib — для визуализации и анимации
- Docker — для контейнеризации
### Локальный запуск

```bash
git clone https://github.com/Milana-gg/praktishh.git

cd praktish
pip install -r requirements.txt
python "Мухина_Милана_БИС-24-2.py"
```
